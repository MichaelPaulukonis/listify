ListiFy

Turn group of lines into HTML list

  How it works

Just use some of this patterns to customize your list

  • li(id='item_#', class='list') > span(class='inner')  Apply
  • li(id='item')  Apply
  • li(id='_#') > h1(class='text')  Apply
  • li(id='_#') > a(href='http://www.google.com')  Apply

1  If no Format is specified, you get a simple list
2  You can specify the tabulation using Spaces input.
3  Ignore digits will ignore the first digits.
4  If you include # in your tags it will generate a number sequence.
